Geetashraya (गीताश्रय)

Geetashraya means "Shelter of Songs".  
It is a full stack music streaming platform where users can listen to songs and artists can upload and manage their music.

This project was built to practice full stack development and understand how a real application works from frontend to backend.

Features

User can browse and play songs  
Music player with play and pause  
Trending songs based on play count  
Artist dashboard to upload songs  
Create albums and manage music  
User profile with profile picture

Tech Stack
